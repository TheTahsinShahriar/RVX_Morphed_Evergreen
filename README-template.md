## 🧩 RVX Morphed Evergreen
Ungated RVX Morphed patches for Morphe.

[![Static Badge](https://img.shields.io/badge/GitHub-gray?style=flat-square&logo=github)](https://github.com/TheTahsinShahriar/RVX_Morphed_Evergreen)   [![Static Badge](https://img.shields.io/badge/Profile-gray?style=flat-square&logo=github)](https://github.com/TheTahsinShahriar)

This fork keeps the original target-version documentation for reference, but removes YouTube and YouTube Music version gating. These patches were made for the documented target ranges; Evergreen lets you try newer versions anyway, with no guarantee that patching will succeed or that the patched app will work correctly.

See the repository for usage details and report issues [here](https://github.com/TheTahsinShahriar/RVX_Morphed_Evergreen/issues).
